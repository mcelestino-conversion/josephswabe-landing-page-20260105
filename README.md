# landing page

a modern, minimalist landing page with a hero section, features section, contact form, and footer. built with next.js, typescript, and tailwind css following best practices for seo and analytics.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - React components
- `public/` - Static assets

## Features

- analytics_integration
- contact_form
- responsive_design
- seo_optimization

## Architecture Decisions

- Performance: Using Next.js Image component, dynamic imports, and caching strategies
